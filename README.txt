黑色金属钢铁数据集说明

负责人：fourmi

原始图像个数：25 （参数：1250-850-0.1）

         宽：1920 高：1200

         保存格式：.JPG
         相关文件夹： images labels


标签图像 类别： 0-贴素体（TST 白色） 
               1-贝氏体（BST 灰色）
               2-图标 （TB 绿色）
               3-珠光体（ZGT 黑色）

         保存格式： .png

*保存格式区分大小写

图像扩充方法：
               /*******************1_expansion.py************************************/

               1:更改图像的亮度（0.87，1.07,1.2,0.8）（四）
               2:图像的左右上下翻转及90，180，270（五）
               3:对图像依次按照宽和高两个参数进行拉伸（二）
               4：取图像的四个顶点及其中心点并依次为起点进行尺度（0.85）裁剪（五）
               效果：使图像扩充16倍（25*16) 生成的文件夹：images_expansion labels_expansion
               
               /*******************2_extract_all_images.py***************************/
               效果：提取当前文件夹下的所有图片并顺序存放在新文件夹下
               生成的文件夹：Imgs_all Labels_all
               
               /*******************3_rotate.py***************************************/
               效果：将Imgs_all Labels_all中的图片依次旋转45 135 225  315 使图片数量扩充4倍（25*16*4）
               生成的文件夹：RotatedImgs RotatedLabels
               
               
最终数据集文件夹：Data 原图包括旋转前和旋转后的 标签包括旋转前和旋转后的，最终图片数量为（400+25*16*4）
